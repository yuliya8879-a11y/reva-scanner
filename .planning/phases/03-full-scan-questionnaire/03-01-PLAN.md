1. `async def create_full_scan(self, user_id: int, scan_type: str) -> Scan`:
   - Creates Scan with scan_type=scan_type, status=ScanStatus.collecting.value, answers={}
   - Validates scan_type is "personal" or "business" (raise ValueError otherwise)
   - commit + refresh + return

   - Gets scan, merges {key: value} into existing answers dict (creates new dict if None)
   - IMPORTANT: Use `scan.answers = {**(scan.answers or {}), key: value}` to trigger SQLAlchemy JSONB mutation detection
   - Does NOT change status (stays collecting)
   - commit + refresh + return

3. `async def get_incomplete_scan(self, user_id: int) -> Optional[Scan]`:
   - SELECT scan WHERE user_id=user_id AND status=collecting AND scan_type IN (personal, business) ORDER BY created_at DESC LIMIT 1
   - Returns Scan or None

   - Sets status=ScanStatus.questionnaire_complete.value
