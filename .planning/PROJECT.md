# Reva Scanner — Цифровой сканер бизнеса

## Vision

AI-система, которая сканирует бизнес клиента по цифровому следу и выдаёт детальный разбор: где утекают деньги, где слепые зоны, где owner сам тормозит, как выстроена система. Аналог личной консультации у Юлии Ревы за 3500₽ — автоматизированный и доступный 24/7.

## Target Users

- Предприниматели с запросом "почему нет денег в бизнесе"
- Самозанятые и фрилансеры
- Владельцы малого бизнеса

## Core Value

Получить точный диагностический разбор бизнеса за 15 минут — без личной консультации, с результатом как у астрологического прогноза: конкретно, понятно, actionable.

## What Gets Scanned

1. **Ответы на вопросы** — структурированная анкета по бизнесу (команда, процессы, деньги, мышление)
2. **Дата рождения / нумерология** — цифровой архетип owner'а, его зоны торможения
3. **Описание ситуации** — свободный текст, AI ищет скрытые паттерны
4. **Соцсети / сайт бизнеса** — анализ публичного цифрового следа

## What Scanner Outputs

- Архитектура бизнеса: где нарушена структура
- Слепые зоны: что owner не видит
- Энергетические блоки: где сам тормозит мыслями/решениями
- Команда: кто не на своём месте
- Деньги: конкретные дыры где утекают
- Рекомендации: что добавить / убрать

## Platforms

- **Telegram-бот** — основная точка входа (воронка из @Reva_mentor)
- **Веб-приложение** — расширенный кабинет, история сканирований

## Monetization

- **Разовая сессия:** 3500₽ за полный разбор
- **Подписка:** безлимитный доступ к боту за 2900₽/мес
- **Воронка:** TG-канал @Reva_mentor → бесплатный мини-скан → платный полный разбор

## TG Channel (@Reva_mentor)

Нужно автоматизировать контент:
- Посты и тексты (регулярный контент)
- Скрипты для Reels
- Контент-план на месяц
- Автопостинг по расписанию

## Technical Stack (предварительно)

- **Telegram Bot:** Python (aiogram) или Node.js (telegraf)
- **AI:** Claude API (Anthropic) — анализ и генерация разборов
- **База данных:** PostgreSQL — история клиентов и сканирований
- **Оплата:** ЮKassa (основной и единственный рабочий канал v1)
- **Веб:** Next.js — личный кабинет клиента
- **Автопостинг:** n8n или python-telegram-bot

## Constraints

- Бюджет: минимальный — начинаем с бота, потом веб
- Монетизация должна работать с первой версии
- Максимально автономная система (минимум ручного участия Юлии)
