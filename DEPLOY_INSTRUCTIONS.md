## ШАГИ ДЕПЛОЯ (порядок важен!)

### ШАГ 1 — Добавить второй API ключ на Railway

1. Открыть [railway.app](https://railway.app) → твой проект
2. Вкладка **Variables**
3. Добавить переменную:
   ```
   ANTHROPIC_API_KEY_2 = [второй ключ Anthropic]
   ```
4. Нажать **Save**

### ШАГ 2 — Смержить ветку в main

В терминале:
```bash
cd ~/reva-scanner
git checkout main
git merge feature/dual-api-access-management
git push origin main
```

Railway автоматически подтянет изменения и перезапустит бота.

### ШАГ 3 — Проверить работу бота

1. Открыть @Eye888888_bot
2. Нажать `/start`
3. Проверить: появилась кнопка `🗄 Мой кабинет 🔒`
4. Проверить: цены на кнопках показываются (3 500 ₽ / 10 000 ₽)
5. Проверить: `/api` в панели Юлии показывает статус двух ключей

### ШАГ 4 — Остановить локальный бот (если ещё работает)

**Проблема**: если на твоём Mac работает локальный `run_polling.py` — он конкурирует с Railway и оба падают.

Найти и убить локальный процесс:
```bash
ps aux | grep run_polling
kill [PID процесса]
```

или просто закрыть терминал где он запущен.
